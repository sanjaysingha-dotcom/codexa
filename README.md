# myCodexa Landing Page

A responsive replica of the myCodexa.ai website built with HTML, CSS, and JavaScript.

## Project Structure

```
landing_page/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Main stylesheet with responsive design
├── js/
│   └── script.js      # JavaScript for interactivity
├── images/            # Image assets (add your images here)
└── README.md          # Project documentation
```

## Features

- **Fully Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and professional design matching the original website
- **Interactive Elements**: 
  - Mobile hamburger menu
  - FAQ accordion
  - Smooth scrolling
  - Form validation
- **All Sections Included**:
  - Hero section
  - Features
  - Journey steps
  - Family planning
  - Health risks
  - Corporate benefits
  - FAQs
  - Insights/Blog
  - Events
  - Registration form
  - Footer

## Getting Started

1. Open `index.html` in your web browser
2. No build process required - pure HTML/CSS/JS

## Customization

- Replace image placeholders in the HTML with actual images
- Update colors in `css/style.css` using CSS variables
- Modify form submission logic in `js/script.js` to connect to your backend

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
